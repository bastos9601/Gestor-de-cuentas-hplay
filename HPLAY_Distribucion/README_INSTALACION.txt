🎬 GESTOR DE CUENTAS STREAMING HPLAY
========================================

📋 DESCRIPCIÓN
---------------
Aplicación profesional para gestionar cuentas de streaming de forma segura.
Incluye sistema de login, gestión de cuentas, notificaciones por WhatsApp y verificación automática de vencimientos.

🚀 INSTALACIÓN
---------------
1. Copia toda la carpeta "HPLAY_Distribucion" a tu dispositivo
2. Asegúrate de que todos los archivos estén en la misma carpeta
3. Haz doble clic en "GestorCuentasHPLAY.exe" para ejecutar

⚠️ IMPORTANTE: No muevas ni elimines ninguno de los archivos de la carpeta

🔐 ACCESO AL SISTEMA
---------------------
- Clave por defecto: admin123
- Puedes cambiar la clave desde la aplicación
- La clave se guarda en config.json

📁 ARCHIVOS INCLUIDOS
---------------------
- GestorCuentasHPLAY.exe (Aplicación principal)
- config.json (Configuración y clave de acceso)
- clave.key (Clave de encriptación)
- cuentas.enc (Base de datos encriptada de cuentas)
- notificaciones.json (Historial de notificaciones)
- PyWhatKit_DB.txt (Base de datos de WhatsApp)
- README_WHATSAPP.md (Documentación de WhatsApp)

🔄 ACTUALIZACIONES
------------------
Para actualizar la aplicación:
1. Descarga la nueva versión
2. Reemplaza solo el archivo .exe
3. Mantén todos los archivos de datos

💡 FUNCIONES PRINCIPALES
------------------------
✅ Gestión de cuentas de streaming
✅ Sistema de login seguro
✅ Búsqueda y filtrado de cuentas
✅ Notificaciones por WhatsApp
✅ Verificación automática de vencimientos
✅ Estadísticas en tiempo real
✅ Interfaz moderna y fácil de usar

🆘 SOLUCIÓN DE PROBLEMAS
-------------------------
Si la aplicación no funciona:
1. Verifica que todos los archivos estén presentes
2. Asegúrate de que no haya antivirus bloqueando
3. Ejecuta como administrador si es necesario
4. Verifica que Windows esté actualizado
